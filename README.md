## Description

Application deployed in heroku and can be accessible here.
https://limehome-booking.herokuapp.com/

```
Framework : NestJS
API Doc : Swagger
DB : Atlas MongoDB
Hotels : Google Places API
Testing : Jest
Deployment: Heroku
```

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
# Need to configure google api key .env file
$ yarn run test:e2e
```
